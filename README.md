# Iris Flower Classification

This is a simple Artificial Intelligence / Machine Learning project built using Python and Scikit-Learn to classify Iris flower species.

## Model Details
- **Algorithm Used:** RandomForestClassifier
- **Dataset:** Built-in Iris Dataset
- **Accuracy Obtained:** 1.0 (100%)

## How to Run
You can run this script using Python 3 or mobile applications like Pydroid 3.
